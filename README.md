# Interview Management System

🚧 Work in Progress

## 📌 Project Description
This is an Interview Management System that helps manage candidate details and interview processes.

## 🚀 Features
- Candidate Registration
- Data Management
- Basic UI using React

## 🛠️ Tech Stack
- React.js
- JavaScript
- HTML
- CSS

## 🔮 Future Improvements

- Interview Scheduling
- Database Integration

## ▶️ How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
